# Parking reservation 
> Frontend portion of Parking reservation web app

# Running locally:

```
> git clone https://github.com/vnbhattarai/parkapp-frontend

> cd parkapp-frontend

> yarn install

> yarn start
```

Then navigate to <http://localhost:3000>
